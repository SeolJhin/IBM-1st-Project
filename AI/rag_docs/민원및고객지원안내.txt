UNI PLACE 민원 및 고객 지원 안내

== 지원 채널 종류 ==
1. 민원 접수 (Complain): 시설 불편, 소음, 수리 요청 등 건물 관련 불편 신고
2. 1:1 문의 (QnA): 계약, 결제, 서비스 이용 관련 개인 문의
3. FAQ: 자주 묻는 질문 모음 (/support/faq)
4. 공지사항: 중요 안내 사항 (/support/notice)

== 민원 접수 ==
- 대상: 활성 계약(contract_st=active) 보유 입주자
- 경로: /support/complain/write
- 접수 가능 항목:
  · 시설 고장 및 수리 요청 (에어컨, 보일러, 수도, 전기 등)
  · 소음·냄새 등 이웃 관련 불편
  · 공용 시설 불편 및 개선 요청
  · 청소·위생 관련 불편
  · 기타 건물 생활 관련 불편 사항

== 민원 처리 상태 ==
- pending(접수 대기): 민원이 접수됨
- received(접수 확인): 관리사무소에서 확인
- in_progress(처리 중): 현재 처리 진행 중
- resolved(해결 완료): 처리 완료, 결과 안내
- closed(종료): 민원 종결

== 민원 처리 기간 ==
- 긴급(high priority): 24시간 이내 처리 (누수, 가스, 화재 관련)
- 일반(medium priority): 3 영업일 이내 처리
- 개선 요청(low priority): 7 영업일 이내 처리

== 민원 우선순위 분류 ==
AI가 민원 내용을 분석하여 우선순위를 자동 분류합니다.
- high(높음): 안전·생활에 즉각적인 위협 (누수, 가스 냄새, 전기 합선 등)
- medium(보통): 생활 불편이 있으나 즉각적 위험 없음 (에어컨 고장, 소음 등)
- low(낮음): 개선 요청, 건의 사항

== 1:1 문의 (QnA) ==
- 대상: 모든 회원 (입주자 우선 처리)
- 경로: /support/qna/write
- 이용 가능 항목:
  · 계약 관련 문의 (재계약, 해지, 변경)
  · 결제·환불 관련 문의
  · 룸서비스 주문 관련 문의
  · 계정·앱 이용 문의
  · 기타 개인 문의

== 문의 처리 상태 ==
- waiting(답변 대기): 문의가 접수됨
- complete(답변 완료): 담당자 답변 완료

== 문의 처리 기간 ==
- 평균 답변 시간: 1~2 영업일
- 긴급 문의는 민원 접수를 함께 이용해 주세요.

== 내 민원·문의 조회 ==
- 민원 목록: /support/complain
- 문의 목록: /support/qna
- AI 챗봇에 "내 민원 보기" 또는 "내 문의 조회"라고 문의하면 해당 페이지로 안내

== 공지사항 ==
- 최신 공지 확인: /support/notice
- 중요 공지는 앱 알림으로도 발송됩니다.
- AI 챗봇에 "최신 공지 알려줘"라고 하면 최근 3개 공지를 바로 안내해드립니다.
