UNI PLACE 결제 및 환불 정책

== 결제 수단 ==
- 신용카드 (국내외 카드 모두 지원)
- 체크카드
- 카카오페이
- 기타 PG사 연동 간편결제

== 결제 항목 ==
- 월세 납부 (매월 지정일)
- 관리비 납부 (월세와 합산 청구 또는 별도)
- 룸서비스 상품 주문
- 공용 시설 유료 이용료 (해당 시설에 한함)

== 월세 결제 ==
- 결제 경로: 마이페이지 → 납부 관리(/me?tab=myroom&sub=rent-payment)
- 결제 시기: 계약 시 지정된 납부일 전후로 알림 발송
- 자동 이체: 현재 수동 납부만 지원 (자동이체 기능 추후 지원 예정)
- 납부 확인서: 마이페이지 → 납부 내역에서 확인 가능

== 룸서비스 결제 ==
- 주문 시 즉시 결제
- 재고 부족·주문 취소 시 즉시 자동 환불 (카드사 기준 3~5 영업일 소요)

== 환불 기준 ==

[월세 환불]
- 이미 납부한 월세의 환불은 원칙적으로 불가
- 단, 건물 측 귀책 사유(시설 하자, 강제 퇴실 등)로 인한 퇴실 시 남은 기간 월세 일할 환불
- 환불 신청: 1:1 문의(/support/qna/write) 또는 관리사무소 방문

[룸서비스 환불]
- PENDING(접수) 또는 CONFIRMED(확인) 상태: 전액 즉시 환불 가능
- PREPARING(준비 중) 이후: 취소 불가
- 상품 불량·오배송: 민원 접수 후 전액 환불 또는 재배송

[보증금 환불]
- 퇴실 후 방 점검 완료 시 14일 이내 반환
- 공제 항목: 방 파손 수리비, 청소비, 미납 월세
- 공제 내역은 퇴실 점검 시 서면으로 안내

== 결제 오류 대처 ==
- 중복 결제: 1:1 문의 또는 민원 접수로 신고 → 확인 후 중복분 즉시 환불
- 결제 실패 후 금액 차감: 카드사로 먼저 확인 → 해결 안 되면 1:1 문의
- 결제 내역 불일치: 마이페이지 → 결제 내역 확인 후 1:1 문의

== 결제 취소(환불) 신청 방법 ==
1. 마이페이지 → 결제 내역 → 해당 건 → 취소 신청
2. 또는 1:1 문의(/support/qna/write)에서 결제 취소 요청
3. 처리 기간: 접수 후 1~3 영업일 내 확인, 환불은 카드사 기준 3~5 영업일

== 현금영수증 및 세금계산서 ==
- 현금영수증: 결제 시 자동 발급 (현금 결제에 한함)
- 세금계산서: 사업자 명의 계약의 경우 발급 가능, 관리사무소 문의
- 월세 소득공제: 무주택 세대주 근로자는 월세 소득공제 가능, 납부 확인서 발급 요청

== 결제 관련 문의 ==
- 1:1 문의: /support/qna/write
- 민원 접수: /support/complain/write
- 결제 내역 조회: 마이페이지 → 결제 내역
