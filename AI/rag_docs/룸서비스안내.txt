UNI PLACE 룸서비스 이용 안내

== 룸서비스란 ==
입주자가 앱에서 생활용품, 청소용품, 식료품 등 필요한 상품을 주문하면 건물 내 또는 지정 장소로 배달해드리는 서비스입니다.

== 이용 자격 ==
- 활성 계약(contract_st=active)이 있는 입주자만 이용 가능합니다.
- 본인이 거주하는 건물에서 재고가 있는 상품만 주문 가능합니다.

== 주문 방법 ==
1. 앱 내 마이페이지 → 룸서비스(/me?tab=roomservice) 접속
2. 건물별 상품 목록 확인 (재고 0개 상품은 주문 불가)
3. 원하는 상품 선택 후 수량 입력
4. 결제 (카드, 카카오페이 지원)
5. 주문 확인 및 배달 대기

== 주문 상태 ==
- PENDING(주문 접수): 주문이 접수되어 처리 대기 중
- CONFIRMED(확인 완료): 관리사무소에서 주문 확인
- PREPARING(준비 중): 상품 준비 중
- DELIVERED(배달 완료): 배달 완료
- CANCELLED(취소): 주문 취소

== 배달 시간 ==
- 배달 가능 시간: 오전 9시 ~ 오후 9시
- 평균 배달 소요 시간: 30분 ~ 1시간 (재고 상황 및 시간대에 따라 상이)
- 주문 폭주 시 지연될 수 있으며, 앱 내 상태 확인 가능

== 결제 및 환불 ==
- 결제 수단: 신용카드, 체크카드, 카카오페이
- 결제는 주문 시 즉시 청구됩니다.
- 환불(취소): PREPARING 상태 이전(PENDING, CONFIRMED)에만 취소 가능
- 취소 방법: 마이페이지 → 룸서비스 → 주문 내역 → 취소
- 배달 완료 후 취소 불가 (상품 불량의 경우 민원 접수 후 처리)

== 상품 불량·오배송 ==
- 상품에 문제가 있을 경우 민원 접수(/support/complain/write)에 사진과 함께 신고
- 오배송은 접수 즉시 재배송 또는 환불 처리

== 건물별 재고 확인 ==
- AI 챗봇에 "우리 건물 룸서비스 상품 있어요?" 또는 "Uniplace A 재고 알려줘"라고 문의
- 품절 상품은 재고 보충 후 주문 가능하며, 일정은 관리사무소에 문의

== 자주 주문하는 상품 ==
AI 챗봇에 "룸서비스 추천해줘"라고 하면 본인의 구매 이력을 기반으로 자주 구매한 상품을 추천해드립니다.
구매 이력이 없는 경우 건물 인기 상품을 추천해드립니다.

== 문의 ==
룸서비스 관련 문의는 1:1 문의(/support/qna/write) 또는 민원 접수(/support/complain/write)를 이용해 주세요.
