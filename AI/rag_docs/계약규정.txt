UNI PLACE 계약 규정

== 계약 유형 ==
- 월세(monthly_rent): 일반 월 단위 임대 계약
- 단기 숙박(stay): 단기 체류 목적의 임대

== 계약 상태 ==
- requested(신청됨): 입주 신청 완료, 관리자 검토 대기
- approved(승인됨): 관리자 승인 완료, 입주 준비 중
- active(활성): 현재 입주 중인 유효 계약
- ended(만료): 계약 기간 종료
- cancelled(해지): 중도 해지된 계약

== 계약 신청 절차 ==
1. 원하는 방 선택 (방 목록 → 방 상세 페이지)
2. 계약 신청 버튼 클릭 → 계약 신청 폼 작성
3. 보증금, 월세, 관리비, 납부일 확인
4. 신청 완료 후 관리자 승인 대기 (보통 1~3 영업일)
5. 승인 후 입주 가능

== 계약 기간 ==
- 기본 계약 기간: 6개월 또는 12개월
- 최소 임대 기간은 방마다 다름 (방 상세에서 확인)
- 계약 만료 30일 전 재계약 여부 안내

== 재계약 ==
- 계약 만료 30일 전부터 재계약 신청 가능
- 재계약 시 월세 조정이 있을 수 있음
- 재계약 의사가 없으면 만료일까지 관리사무소에 통보 필요
- 재계약은 1:1 문의(/support/qna/write) 또는 관리사무소 방문

== 계약 해지(중도 해지) ==
- 계약 해지 시 반드시 30일 전 사전 통보 필요
- 사전 통보 없이 퇴실 시 위약금 발생 가능
- 해지 신청: 1:1 문의 또는 관리사무소 방문
- 보증금 반환: 퇴실 후 방 점검 완료 후 14일 이내 환불
- 방 파손·청소 미이행 시 보증금에서 공제 가능

== 위약금 기준 ==
- 계약 시작 1개월 이내 해지: 보증금의 10%
- 계약 기간 절반 미만 경과 후 해지: 남은 기간 월세의 1개월분
- 계약 기간 절반 이상 경과 후 해지: 위약금 없음 (단, 30일 사전 통보 필수)

== 보증금 ==
- 보증금은 입주 전 납부
- 계약 종료 후 방 점검 완료 시 반환
- 방 파손, 미납 월세, 청소 비용 등 공제 후 반환 가능
- 반환 기간: 퇴실 후 최대 14일

== 월세 및 관리비 납부 ==
- 납부일: 계약 시 지정된 날짜 (예: 매월 25일)
- 납부 방법: 앱 내 결제 (/me?tab=myroom&sub=rent-payment)
- 관리비: 공용 시설 유지·관리비, 인터넷, 수도, 공용 전기 포함
- 개별 전기·가스는 별도 청구될 수 있음

== 연체 규정 ==
- 납부일 기준 7일 초과 연체 시 연체료 발생
- 연체료: 미납 금액의 1% (일 단위)
- 2회 이상 연속 연체 시 계약 해지 통보 가능
- 미납 내역 확인: 앱 내 마이페이지 → 납부 관리

== 계약서 확인 ==
- 계약서는 앱 내 마이페이지에서 PDF로 확인 가능
- 경로: /me?tab=myroom&sub=contracts
