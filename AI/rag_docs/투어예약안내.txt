UNI PLACE 사전 방문(투어) 예약 안내

== 투어란 ==
계약 전에 입주를 희망하는 방을 직접 방문해 확인하는 서비스입니다.
방 상태, 크기, 옵션, 건물 분위기 등을 눈으로 확인할 수 있습니다.

== 투어 예약 방법 ==
1. AI 챗봇에 "방 투어 예약하고 싶어요"라고 말하거나 예약 페이지(/reservations/tour/create) 직접 방문
2. 원하는 건물 선택
3. 보고 싶은 방 선택 (available 상태인 방만 투어 가능)
4. 방문 희망 날짜 선택
5. 예약 완료 → 이름, 연락처 입력

== 투어 가능 시간 ==
- 평일(월~금): 오전 10시 ~ 오후 6시
- 주말·공휴일: 관리사무소와 사전 협의 필요

== 투어 상태 ==
- pending(대기 중): 예약 접수, 확정 대기
- confirmed(확정): 투어 일정 확정
- cancelled(취소): 취소된 예약
- completed(완료): 투어 완료
- no_show(노쇼): 예약 후 미방문

== 투어 취소 ==
- 투어 시작 24시간 전까지 무료 취소 가능
- 취소 방법: 마이페이지 → 투어 예약 목록 → 취소
- 또는 AI 챗봇 → "내 투어 예약 보기" → 목록 페이지에서 취소

== 노쇼(미방문) 안내 ==
- 예약 후 연락 없이 미방문 시 노쇼 처리됩니다.
- 노쇼 2회 이상 시 투어 예약이 일시 제한될 수 있습니다.
- 방문이 어려울 경우 반드시 사전에 취소해 주세요.

== 투어 당일 안내 ==
- 신분증 지참 필수 (본인 확인 용도)
- 예약 시 기재한 연락처로 관리자가 사전 연락드릴 수 있습니다.
- 지각 시 관리사무소로 연락해 주세요.

== 투어 후 계약 신청 ==
투어 후 마음에 드는 방이 있다면 바로 계약 신청이 가능합니다.
- 방 상세 페이지 → 계약 신청 버튼
- 또는 AI 챗봇에 "이 방 계약 신청하고 싶어요"라고 문의

== 비대면 투어 ==
직접 방문이 어려운 경우 사진, 영상 자료를 요청할 수 있습니다.
1:1 문의(/support/qna/write)로 희망 방 정보와 함께 요청해 주세요.

== 내 투어 예약 조회 ==
- 예약 목록 확인: /reservations/tour/list
- AI 챗봇에 "내 투어 예약 확인하고 싶어요"라고 문의하면 바로 연결됩니다.
