UNI PLACE 커뮤니티 이용 규정

== 커뮤니티 종류 ==
- 자유 게시판: 입주자 간 자유로운 소통 공간
- 리뷰 게시판: 방·건물 이용 후기 작성
- 공지사항: 관리자 공지 (읽기만 가능)

== 이용 자격 ==
- 자유 게시판: 로그인한 모든 회원
- 리뷰 게시판: 해당 방에 계약 이력이 있는 입주자
- 공지사항: 모든 회원 (비회원 포함) 읽기 가능

== 게시글 작성 규칙 ==
다음에 해당하는 내용은 게시 금지입니다.
- 타인을 비방, 모욕, 명예 훼손하는 내용
- 개인정보(연락처, 주소 등) 포함 게시물
- 광고, 홍보, 스팸 성격의 게시물
- 허위 사실 유포
- 음란·폭력·혐오 표현
- 정치적 주장, 종교 갈등 유발 내용
- 저작권 침해 콘텐츠 (타인 사진, 글 무단 사용)

== 익명 게시 ==
- 자유 게시판에서 익명 게시 선택 가능
- 익명이어도 관리자는 작성자를 확인할 수 있습니다.
- 익명을 악용한 비방·욕설 게시 시 제재 가능

== 게시물 삭제 기준 ==
다음의 경우 관리자가 게시물을 삭제할 수 있습니다.
- 이용 규정 위반 게시물
- 다수의 신고를 받은 게시물
- 개인정보 노출 게시물

== 제재 기준 ==
- 1회 위반: 해당 게시물 삭제 및 경고
- 2회 위반: 7일 커뮤니티 이용 제한
- 3회 이상: 30일 이용 제한 또는 영구 정지

== 리뷰 작성 규칙 ==
- 실제 입주 경험에 기반한 솔직한 리뷰 작성
- 허위 리뷰, 비방 목적 리뷰는 삭제됩니다.
- 별점은 1~5점으로 평가
- 리뷰 수정은 작성 후 7일 이내만 가능

== 좋아요·댓글 ==
- 게시물에 좋아요 및 댓글 작성 가능
- 로그인한 회원만 가능
- 댓글도 게시글과 동일한 규정 적용

== 내 게시글 조회 ==
- 마이페이지 → 나의 활동 → 내가 쓴 글
- AI 챗봇에 "내가 쓴 글 보기"라고 문의하면 바로 연결됩니다.

== 신고 방법 ==
부적절한 게시물 발견 시 게시물 신고 기능을 이용하거나 1:1 문의(/support/qna/write)로 신고해 주세요.
