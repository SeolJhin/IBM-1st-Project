UNI PLACE 반려동물 입주 규정

== 기본 원칙 ==
반려동물 허용 여부는 방(room)마다 개별 설정되어 있습니다.
방 목록 또는 방 상세 페이지에서 "반려동물 허용" 여부를 반드시 확인하세요.
반려동물 허용(pet_allowed_yn=Y) 방에서만 동반 입주가 가능합니다.

== 허용 반려동물 ==
- 소형견 (체중 7kg 이하)
- 고양이 (마릿수 제한: 2마리 이하)
- 소형 설치류 (햄스터, 기니피그 등)
- 관상어 (어항 크기 30L 이하)

== 허용 불가 반려동물 ==
- 중형·대형견 (체중 7kg 초과)
- 파충류, 조류 (소음 및 위생 문제)
- 위험 동물 (법적 허가 필요 동물 포함)

== 입주 전 신고 의무 ==
- 반려동물 동반 입주 시 계약 신청 단계에서 반드시 고지해야 합니다.
- 반려동물 종류, 품종, 체중을 관리사무소에 사전 통보해야 합니다.
- 미고지 후 반려동물 발견 시 계약 해지 사유가 될 수 있습니다.

== 입주자 준수 사항 ==
- 엘리베이터 탑승 시 반드시 케이지 또는 이동장 사용
- 공용 공간(라운지, 헬스장, 회의실 등)에 반려동물 출입 금지
- 반려동물 배변은 반드시 실내에서 처리, 공용 공간 배변 엄금
- 심한 짖음·소음으로 타 입주자 민원 발생 시 시정 조치
- 반려동물에 의한 방 파손(벽지, 바닥재 등)은 퇴실 시 원상복구 또는 비용 부담

== 청소 및 위생 ==
- 반려동물 털, 냄새 등 관리 책임은 입주자에게 있습니다.
- 퇴실 시 반려동물로 인한 오염·파손이 있을 경우 청소비 및 수리비가 보증금에서 공제됩니다.
- 청소비 기준: 전문 청소 필요 시 실비 청구 (보통 10만~30만 원)

== 타 입주자 피해 발생 시 ==
- 반려동물 소음·냄새 등으로 2회 이상 민원 접수 시 경고
- 3회 이상 반복 민원 시 계약 해지 통보 가능

== 반려동물 허용 방 찾기 ==
- 방 목록 페이지(/rooms)에서 "반려동물 허용" 필터 선택
- AI 챗봇에 "반려동물 키울 수 있는 방 있나요?" 또는 "반려동물 허용 방 알려줘" 문의
