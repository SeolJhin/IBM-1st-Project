UNI PLACE 방 및 건물 안내

== 건물 종류 ==
현재 운영 중인 건물: Uniplace A, Uniplace B, Uniplace C
각 건물은 위치, 편의 시설, 주차 가능 여부 등이 다릅니다.
건물 목록 및 상세 정보: /buildings

== 방 타입 ==
- 원룸(one_room): 주방·거실·침실 통합, 1인 거주 적합
- 투룸(two_room): 방 2개, 1~2인 거주 적합
- 쓰리룸(three_room): 방 3개, 2~3인 거주 적합
- 복층(loft): 복층 구조, 독특한 공간 활용
- 쉐어(share): 공용 주방·거실 공유, 개인 침실 독립

== 방 상태 ==
- available(입주 가능): 현재 입주 가능
- reserved(예약됨): 계약 신청 처리 중
- contracted(계약 중): 입주자 거주 중
- repair(점검·수리 중): 입주 불가
- cleaning(청소 중): 퇴실 후 청소 중, 곧 입주 가능

== 방 옵션 (기본 제공) ==
방마다 제공 옵션이 다릅니다. 방 상세 페이지에서 확인하세요.
- 에어컨(aircon)
- 책상(desk)
- 냉장고(refrigerator)
- 세탁기(washer)
- 침대(bed)
- 옷장(wardrobe)
- 전자레인지(microwave)
- TV

== 방 검색 방법 ==
- 방 목록 페이지(/rooms)에서 조건 검색
- 조건: 건물, 방 타입, 보증금, 월세 범위, 반려동물 허용, 옵션 등
- AI 챗봇에 조건을 말하면 맞는 방을 찾아드립니다.
  예: "보증금 없는 원룸 있나요?", "에어컨 있는 방 알려줘", "반려동물 키울 수 있는 방"

== 월세·관리비 구조 ==
- 월세(rent_price): 방 임대료
- 관리비(manage_fee): 공용 시설, 인터넷, 공용 전기·수도 포함
- 보증금(deposit): 입주 시 납부, 퇴실 시 반환 (0원 보증금 방도 있음)
- 납부일: 계약 시 지정 (예: 매월 25일)

== 임대 유형 ==
- 월세(monthly_rent): 일반 월 단위 계약
- 단기 숙박(stay): 단기 체류 목적

== 건물 편의 시설 ==
건물별로 다음 편의 시설이 제공될 수 있습니다.
- 엘리베이터
- 주차장
- 헬스장(Gym)
- 라운지(Lounge)
- 회의실(Meeting Room)
- 스터디룸(Study Room)
공용 시설 목록 및 예약: /spaces

== 방 향(방향) ==
- 동향: 오전 햇빛, 오후 그늘
- 서향: 오후 햇빛 (오후에 따뜻함)
- 남향: 하루 종일 햇빛 (가장 선호)
- 북향: 직사광선 없음, 상대적으로 시원

== 투어(사전 방문) 예약 ==
계약 전 방을 직접 확인하고 싶다면 투어를 예약하세요.
- 예약 경로: /reservations/tour/create
- AI 챗봇에 "방 투어 예약하고 싶어요"라고 하면 단계별로 안내해드립니다.
- 투어는 평일 오전 10시 ~ 오후 6시 사이 진행

== 리뷰 ==
- 입주 경험이 있는 입주자는 방·건물 리뷰를 작성할 수 있습니다.
- 리뷰 확인: 방 상세 페이지 또는 커뮤니티(/community?tab=REVIEW)
- 리뷰 작성: 마이페이지 → 나의 활동 → 리뷰 작성
