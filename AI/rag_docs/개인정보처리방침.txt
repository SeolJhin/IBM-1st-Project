UNI PLACE 개인정보 처리방침

== 수집하는 개인정보 항목 ==

[회원 가입 시]
- 필수: 이름, 이메일, 비밀번호, 연락처
- 선택: 프로필 사진

[소셜 로그인 시]
- 카카오: 카카오 계정 이메일, 닉네임, 프로필 사진 (카카오 정책에 따름)

[계약 체결 시]
- 주민등록번호 (본인 확인 목적, 수집 후 즉시 암호화)
- 은행 계좌 정보 (보증금 반환 목적)

[서비스 이용 중]
- 이용 기록, 접속 로그, 결제 내역
- 민원·문의 내용

== 개인정보 수집 목적 ==
- 회원 식별 및 계정 관리
- 계약 체결 및 이행
- 결제 처리 및 청구
- 고객 지원 및 민원 처리
- 서비스 개선 및 맞춤형 서비스 제공
- 법적 의무 이행

== 개인정보 보관 기간 ==
- 회원 정보: 탈퇴 후 즉시 삭제 (단, 법적 보관 의무 항목 제외)
- 계약 관련 정보: 계약 종료 후 5년 (전자상거래법)
- 결제 정보: 5년 (전자상거래법)
- 민원·문의 내용: 3년

== 개인정보 제3자 제공 ==
원칙적으로 제3자에게 개인정보를 제공하지 않습니다.
단, 다음의 경우 예외적으로 제공합니다.
- 법령에 의한 요청 (수사기관, 법원 등)
- 결제 처리를 위한 PG사 제공 (결제 정보에 한함)

== 개인정보 처리 위탁 ==
- 결제 처리: 카카오페이, PG사
- 서버 운영: 클라우드 서비스 제공업체
위탁 업체에는 개인정보 보호 의무를 계약으로 부과합니다.

== 이용자 권리 ==
이용자는 언제든지 다음 권리를 행사할 수 있습니다.
- 개인정보 열람 요청
- 오류 정정 요청
- 삭제 요청 (단, 법적 보관 의무 항목 제외)
- 처리 정지 요청

권리 행사 방법: 1:1 문의(/support/qna/write) 또는 관리사무소 방문

== 쿠키 및 자동 수집 정보 ==
- 서비스 이용 편의를 위해 쿠키를 사용합니다.
- 쿠키 거부 시 일부 서비스 이용이 제한될 수 있습니다.
- 브라우저 설정에서 쿠키를 거부할 수 있습니다.

== 개인정보 보호책임자 ==
- 문의: 1:1 문의(/support/qna/write)
- 개인정보 침해 신고: 한국인터넷진흥원(KISA) privacy.kisa.or.kr

== 방침 변경 ==
개인정보 처리방침이 변경되면 앱 내 공지사항(/support/notice)을 통해 안내드립니다.
