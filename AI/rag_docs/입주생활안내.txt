UNI PLACE 입주 생활 안내

== 입주 첫날 체크리스트 ==
1. 관리사무소에서 출입 카드/키 수령
2. 방 상태 점검 (파손·하자 사진 촬영 및 신고)
3. 인터넷 공유기 위치 및 비밀번호 확인
4. 쓰레기 분리수거 장소·요일 확인
5. 비상 연락처 (관리사무소, 경비실) 저장
6. 앱 로그인 및 계약 정보 확인

== 하자 신고 ==
- 입주 후 7일 이내 발견된 기존 하자는 반드시 신고해야 합니다.
- 신고 방법: 민원 접수(/support/complain/write) → 사진 첨부
- 신고하지 않은 하자는 퇴실 시 입주자 과실로 처리될 수 있습니다.

== 쓰레기 분리수거 ==
- 일반 쓰레기: 지정 종량제 봉투 사용 (편의점·마트 구매)
- 재활용: 건물 내 분리수거함에 종류별 분리 배출
  - 종이류, 플라스틱, 유리, 캔, 스티로폼 분리
- 음식물 쓰레기: 건물 내 음식물 쓰레기통 이용 (계량기 포함 건물 별도 안내)
- 대형 폐기물: 관리사무소에 사전 신청 후 배출 (별도 수수료)
- 배출 시간: 건물별 상이 (관리사무소 안내 참고)

== 소음 기준 ==
- 심야 시간(오후 11시 ~ 오전 7시) 층간 소음, 큰 음악 소리 금지
- 실내 악기 연주는 오전 10시 ~ 오후 9시에만 허용 (건물별 상이)
- 소음으로 인한 민원 접수 시 경고 및 시정 요청

== 흡연 ==
- 건물 내부(방, 복도, 엘리베이터, 공용 시설) 전면 금연
- 지정 흡연 구역은 건물 외부에 별도 안내
- 실내 흡연 적발 시 원상복구(도배·청소) 비용 청구 가능

== 방문객 ==
- 방문객은 입주자 동반 하에만 건물 출입 가능
- 외부인 단독 출입 불가 (보안 카드 필요)
- 장기 방문객(7일 이상 상주)은 관리사무소에 사전 신고

== 주차 ==
- 입주자 지정 주차 공간은 계약 시 배정 (일부 건물은 선착순)
- 방문객 주차는 단기(2시간 이내) 허용, 장기 주차는 사전 협의
- 무단 주차 차량은 견인 조치될 수 있습니다.
- 주차 가능 여부·대수는 건물 상세 페이지에서 확인

== 택배 ==
- 택배 수령: 1층 무인 택배함 또는 관리사무소 보관 (건물별 상이)
- 장기 미수령 택배(7일 이상)는 반송될 수 있습니다.
- 냉장·냉동 택배는 수령 당일 가져가야 합니다.

== 인터넷 ==
- 건물 내 무료 와이파이 제공 (관리비 포함)
- 비밀번호는 입주 시 관리사무소에서 제공
- 인터넷 장애 시 민원 접수 또는 관리사무소 신고

== 에어컨·냉난방 ==
- 에어컨·난방기 사용 방법은 방 비치 매뉴얼 참고
- 고장 시 민원 접수(/support/complain/write)에 신고
- 필터 청소는 입주자 직접 또는 요청 시 유료 청소 서비스 제공

== 긴급 상황 대처 ==
- 화재: 즉시 119 신고 → 관리사무소 연락 → 비상구로 대피
- 수도 누수: 수도 밸브 잠금 → 관리사무소 즉시 신고
- 가스 냄새: 창문 개방 → 가스 밸브 잠금 → 119 신고 → 건물 외부로 대피
- 엘리베이터 갇힘: 내부 비상 버튼 누름 → 119 신고

== 퇴실 안내 ==
- 퇴실 30일 전 관리사무소 또는 1:1 문의로 퇴실 의사 통보
- 퇴실 당일: 짐 이동 → 방 청소 → 관리사무소 방 점검 → 키/카드 반납
- 보증금은 점검 완료 후 14일 이내 반환
- 에어컨 필터, 욕실, 바닥 청소 미이행 시 청소비 공제

== 관리사무소 이용 ==
- 운영 시간: 평일 오전 9시 ~ 오후 6시
- 긴급 연락: 24시간 경비실 운영 (건물별 상이)
- 문의 방법: 방문, 전화, 앱 내 1:1 문의(/support/qna/write)
